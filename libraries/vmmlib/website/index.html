VMMLib - Vector and Matrix Math Library

VMMLib is a vector and matrix math library implemented using C++ templates, thus making it very easy to integrate into other libraries and programs. Its BSD license allows the usage both in open source and commercial closed source software. For downloads and further information, please visit the SourceForge project page. If you are using subversion, you can easily integrate the VMMLib into your repository. Since the VMMLib is template-based, there is no need to build or install a library. VMMLib was developed by the Visualization and Multimedia Lab at the University of Zürich, and is currently used by the following projects:
